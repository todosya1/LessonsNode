Модуль url.parse используется для анализа URL
и возвращает объект, содержащий информацию из URL. Пример использования:
console.log(url.parse(pathName, true));

Результат работы url.parse:
path: путь запроса (например, /product).
query: объект, содержащий параметры запроса (если они есть). Например, 
из URL ?name=John будет извлечен объект { name: 'John' }.
Аргумент true в url.parse:

Если передан, то параметры запроса автоматически преобразуются в объект. Без этого аргумента параметры возвращаются в виде строки.
Полный путь запроса: /product?id=1&name=John
Путь: /product
Параметры: { id: '1', name: 'John' }
