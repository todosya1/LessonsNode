Домашнее задание #1: Создание сервера с использованием статического middleware и динамических маршрутов

Описание: Создайте сервер, который обслуживает статические файлы и динамические данные через API. В основном, вам нужно реализовать обработку параметров через URL-запросы.

Требования:

Статическая структура:
Создать директорию assets содержащую:
index.html - главная страница с ссылками на /about и /contact
about.html - страница "О нас"
contact.html - страница с контактной формой
styles.css - стили для всех страниц
logo.png - логотип для добавления
Настроить express.static() для обслуживания директории assets
Динамические маршруты:
GET /products - возвращает список продуктов в JSON
GET /products/:id - возвращает конкретный продукт по ID (если не найден - 404)
GET /users - возвращает список пользователей с фильтрацией по возрасту
Обработка ошибок:
Если пользователь запрашивает несуществующий маршрут, отдавать 404.html из директории assets
